Projets de développement

SAE1.05 - Analayse de données en python

La photo présentée ci-contre représente la réalisation d'un projet mené dans le cadre de mes études. Nous devions, par groupe de 2, imaginer un programme python servant à analyser des données à partir d'un fichier CSV. Nous devions donc collecter des données et imaginer quels traitements pouvaient être intéressants à effectuer sur ces dernières. Dans notre cas, nous avons utilisé des données Spotify

Ce projet m'a donc permis d'apprendre à traiter des données en python en découvrant des bibliothèques importantes comme matplotlib ou tkinter. J'ai également pu apprendre à surmonter des erreurs de code et à travailler en équipe sur un projet de développement.

SAE2.03Base de données et PostgreSQL

La capture présentée ci-contre représente la réalisation d'un site web crée dans le cadre d'un projet scolaire. Nous devions, par groupe de 2, imaginer une base de données que nous devions remplir et interroger et interroger comme nous la souhaitions. Ici, nous avons décidé d'imaginer une base de données regroupent des logs classés par entreprise, machine et niveau de gravité. Le projet ne devait pas aller plus loins mais comme il nous restait du temps nous avons décidé d'implémenter cette interface web pour rendre notre Système de détection d'intrusion plus vivant.

Ce projet m'a donc permis d'apprendre à interroger une base de données à l'aide du langage SQL mais également à lier une base de données à un site web avec PHP. J'ai également pu apprendre à dépanner des erreurs de requêtes et à travailler en équipe sur un projet de développement.

SAE3.02 - Développer des applications utilisant les graphes

La capture présentée ci-contre représente l'affichage d'un programme python réalisé dans le cadre d'un projet scolaire. Nous devions, par groupe de 2, réaliser un programme python utilisé afin de traiter un problème relatif à l'algorithmique des graphes. Dans notre cas, nous avons utilisé ce projet afin d'essayer de trouver un algorithme permettant de trouver une solution au problème de l'arbre de Steiner en trouvant l'arbre couvrant de coût minimal d'un raiseau de laisons WAN

Ce projet m'a donc permis d'apprendre les bases des algorithmiques des graphes au travers du langage python. J'ai également du apprendre à travailler en équipe tout en essayant de réaliser un problème qui n'a aucune solution réelle.
