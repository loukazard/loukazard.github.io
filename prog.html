Analayse de donnéesen python

La photo présentée ci-contre représente la réalisation d'un projet mené dans le cadre de mes études. Nous devions, par groupe de 2, imaginer un programme python servant à analyser des données à partir d'un fichier CSV. Nous devions donc collecter des données et imaginer quels traitements pouvaient être intéressants à effectuer sur ces dernières. Dans notre cas, nous avons utilisé des données Spotify

Ce projet m'a donc permis d'apprendre à traiter des données en python en découvrant des bibliothèques importantes comme matplotlib ou tkinter. J'ai également pu apprendre à dépanner des erreurs de code et à travailler en équipe sur un projet de développement.
